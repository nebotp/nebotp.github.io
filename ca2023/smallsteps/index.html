Explotación


La formula para encriptar la flag es el mensaje original elevado al exponente, el problema aqui es que el exponente es 3 que es demasiado pequeño

message_encryoted = menssage_original ^ exponent

Aplicando la formula inversa para obtener el mensaje original seria la raiz del exponente del mensaje encriptado, en este caso como es 3 seria la raiz cubica

message_original = message_encrypted ^ (1/exponent)

message_original = message_encrypted ^ (1/3)

En un script de python definimos las varibles de el mensaje encriptado y el exponente y aplicamos la formula anterior, y mostramos el resultado en fomato bytes

#!/usr/bin/python3
from Crypto.Util.number import long_to_bytes
from gmpy2 import iroot

message_encrypted = 70407336670535933819674104208890254240063781538460394662998902860952366439176467447947737680952277637330523818962104685553250402512989897886053  
exponent = 3

message_original = iroot(message_encrypted, exponent)[0]
flag = long_to_bytes(message_original)

print(flag.decode())

Al ejecutarlo aplica la formula con los valores y nos muestra la flag desencriptada
